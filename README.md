# RubyCrawler

Um web-crawler escrito em Ruby on Rails.
Veja a demonstração em https://wp-rubycrawller.herokuapp.com/

## Ruby version 
2.5.3

## Rails version
5.2.4

## Configuração
Clone ou realize o download do código. Dentro da pasta do projeto, execute:
```console
$ bundle install
```

## Execução local
```console
$ rails s
```
